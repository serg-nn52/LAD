RPG-баттл

Бой идет по ходам. Каждый ход компьютер (Лютый) случайно выбирает одно из доступных действий и сообщает, что он собирается делать. В ответ на это игрок (Евстафий) должен выбрать свое действие. После происходит взаимное нанесение урона. Магическая броня блокирует магический урон, физическая броня блокирует физический урон. После совершения действия, оно не может быть повторно выбрано в течение cooldown ходов Бой идет до победы одного из противников. Перед началом боя игрок выбирает сложность (начальное здоровье Евстафия)
